1:39 PM — Identified what Brave's first-run wizard configures and scripted the

--------------------------------------------------------------------------

1:48 PM — Disabled Brave's built-in password manager entirely, and
force-installed the iCloud Passwords extension via a mandatory macOS policy
(the one step that needs sudo, since it writes to /Library/Managed Preferences
instead of a user-level plist). Had the script finish by opening
mail.google.com in Brave to sign into Gmail by hand, with the extension there
to autofill from iCloud Keychain once it's signed in too.

--------------------------------------------------------------------------

1:52 PM — Added a Finder default view to macos-defaults.sh: List view, sorted by
Date Added descending. Only affects folders with no per-folder view settings of
their own, which on a fresh account is everything.

--------------------------------------------------------------------------

2:04 PM — Added Clipy to the Brewfile. `brew bundle` covers the install; noted
that it'll ask for Accessibility permission on first launch.

--------------------------------------------------------------------------

2:12–2:13 PM — Replicated Clipy's current preferences (pulled via `defaults read
com.clipy-app.Clipy` on this machine) — numeric key equivalents, beta

and which clipboard types get stored. Skipped the stuff that's just

confirm "cleared" means the key is simply absent rather than an empty value —

wasn't set on this machine to begin with and so wasn't guessable from the
`defaults read` output.

2:40 PM — Looked into scripting Keyboard Maestro's macro sync (target file lives
in iCloud Drive, under a folder named "Google Drive"). No preference key exists

--------------------------------------------------------------------------

2:43 PM — Added a step to disable Tips before any app installs happen. Stops and

widget plist). Reversible with `launchctl enable`.
